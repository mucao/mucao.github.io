2. HTML怎么设置字与字之间的间距代替空格

空格： &nbsp

CSS:letter-spacing字与字
                word-spacing词与词

行距:line-height:1.5;

段落:

<p style="margin-top:30px;text-indent:2em;">文字</p>

字与字之间：

<span style="margin-right:20px;">文字1</span>文字2

文字与单词间距，样式加这个：letter-spacing:20PX (这是字符与字符，对中 英 适用)

单词与单词间距，样式加这个：word-spacing :20PX (这是单词与单词)；

3. MarkDown中打印空格

半角空格(英文) &nbsp;

全角空格(中文) &emsp;

4. 在markdown中进行注释

可以使用下面这种方式：

a

[^_^]:
    b

c

d

显示的 a c d 应当间隔一样长。如下：

a

c

d

其实markdown是兼容html的，所以也可以使用html的注释来进行markdown的注释。
